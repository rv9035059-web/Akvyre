# Editable Akvyre Website

Files:
- index.html → Fully editable main website
- assets/ → All extracted CSS, JS, fonts, images, and bundled files

You can now:
- Edit text directly in index.html
- Modify styles in extracted CSS files
- Change animations and logic in JS files
- Replace assets/images/fonts

Recommended editors:
- VS Code
- Cursor
- WebStorm

To preview locally:
1. Open the folder in VS Code
2. Install "Live Server"
3. Run index.html with Live Server
